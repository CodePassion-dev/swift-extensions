## Declaration: 
> func toWordsArray() -> [String]


## Description: 
> Array of words in a string.


#### Usage Example: 
`````
let string: String = "Hello World!".
print(string.wordCount()) // ["Hello", "World!"]
`````

## Parameters: 
> None


## Returns: 
> The array of words from a given string as an array of string value.
