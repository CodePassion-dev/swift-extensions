## Declaration: 
> func wordCount() -> Int


## Description: 
> Counting of words in a string.


#### Usage Example: 
`````
let string: String = "Hello World!".
print(string.wordCount()) // 2
`````

## Parameters: 
> None


## Returns: 
> Number of words from a given string as an int value.
