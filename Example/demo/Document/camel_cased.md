## Declaration: 
> func camelCased() -> String


## Description: 
> Convert string into camel case.


#### Usage Example: 
`````
let string: String = "Hello world!"
print(string.camelCased()) // helloWorld!
`````

## Parameters: 
> None


## Returns: 
> Camel string value from a given string as a string value.
